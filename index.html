  • Robust Prototypical Few‑Shot Organ Segmentation with Regularized Neural‑ODEs

    ArXiv

    We propose a novel robust few‑shot segmentation framework, Regularized Prototypical Neural Ordinary Differential Equation (R‑PNODE), that provides defence against gradient‑based adversarial attacks by stabilizing Neural‑ODEs with Gaussian perturbations. We employ cluster and consistency losses that allow robustness against input variations and make prototypes (from support) and query features robust.

  • Mathematical Expressions in Software Engineering Artifacts

    6th International Conference on Data Science & Management of Data (CODS‑COMAD’ 23)

    We construct a dataset of bug reports with both automated and manually annotated mathematical expressions. We share a regex based tool MEDSEA to identify and remove mathematical expressions in code changes which show improvement in log message generation tasks like CommitGen.

  • Adversarially Robust Prototypical Few-shot Segmentation with Neural-ODEs

    25th International Conference on Medical Image Computing and Computer Assisted Intervention (MICCAI '22)

    We propose a novel robust few-shot segmentation framework, Prototypical Neural Ordinary Differential Equation (PNODE), that provides defence against gradient-based adversarial attacks. We show that our framework is more robust compared to traditional adversarial defence mechanisms such as adversarial training.
